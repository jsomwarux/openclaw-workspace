## Lesson Format (10 minutes)
1. **Review** (2 min) — 3 quick recall prompts from yesterday's topic
2. **New material** (5 min) — 3–4 key phrases with TTS audio, context, and memory hook
3. **Practice** (2 min) — 2 prompts for JT to respond to (voice message preferred, text OK)
4. **Challenge** (1 min) — today's real-world challenge if applicable

## Evaluation Format (Sunday)
- Role-play scenario (JT speaks, Eve plays the other character)
- 5 vocabulary recall prompts (Spanish → English or fill-in-the-blank)
- Pronunciation feedback on voice responses
- Score: 1–10 | Pass threshold: 6/10 to advance
- If <6: reteach weak areas before advancing week

## Voice Practice Protocol
- JT sends voice message in Telegram with the Spanish phrase
- Eve transcribes via Groq whisper-large-v3 and evaluates:
  - Was the meaning conveyed? ✅/❌
  - Key errors flagged (don't overwhelm — max 2 corrections per response)
  - Encouragement + retry if significantly off
- Text responses accepted as fallback — evaluated the same way
