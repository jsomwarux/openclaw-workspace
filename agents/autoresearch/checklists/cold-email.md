# Autoresearch Checklist — cold-email/SKILL.md
# Message 1 scoring (yes=1, no=0 per question, averaged across test inputs)

1. Is the message under 100 words?
2. Does the opener reference something specific to the INDIVIDUAL (not just company or niche)?
3. Is there no signature block in Message 1?
4. Does the message contain no "I built X" claim?
5. Does the closing question avoid a binary choice structure ("still manual or have you built something?")?
6. Is the opener format one of the 5 approved formats (recent signal, peer-level drop, milestone hook, contrarian observation, curiosity opener)?
