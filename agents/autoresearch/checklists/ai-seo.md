# AI SEO Autoresearch Checklist

1. Does the output build or load a canonical business context before recommending GEO/AI-search fixes?
2. Does it mark unknown facts as unknown and request evidence instead of inventing rankings, citations, profiles, or competitors?
3. Does it prioritize ready-to-hire keywords, entity consistency, structured answers, schema/robots/llms.txt, and citation-worthy proof before broad awareness content?
4. Does it avoid guaranteed ranking/citation claims, spammy directory blasting, fake review language, and knowledge-panel/Wikipedia work for non-notable SMBs?
5. Does it produce a practical operator report or prioritized fix list with owners, evidence needs, and human-approval boundaries for public changes?
6. For local SMB work, does it compare public surfaces and competitor gaps using table-stakes/strong-recommendation/differentiation/optimize/create-fix classifications?
