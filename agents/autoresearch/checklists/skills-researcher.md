# Autoresearch Checklist — skills-researcher

Score skills/API research outputs against these yes/no checks:

1. Does it evaluate usefulness before surfacing a tool to JT?
2. Does it separate hype from practical workflow impact?
3. Does it include cost, setup friction, and security/access risk?
4. Does it recommend skip/add/watch with a clear reason?
5. Does it avoid asking JT to decide before Eve has evaluated the tool?
6. Does it create a task only for tools worth adding or testing?
