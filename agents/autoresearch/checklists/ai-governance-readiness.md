# Autoresearch Checklist — ai-governance-readiness

Target skill: `skills/ai-governance-readiness/SKILL.md`

Use yes/no only. Pass requires all yes.

1. Does the audit return a clear verdict: Ready, Ready after cleanup, or Not ready yet?
2. Does it identify what AI may automate/draft versus what requires human approval or must not be automated?
3. Does it score data readiness, approval path, sensitive-data exposure, logging/auditability, rollback, value, and complexity?
4. Does it list concrete failure modes with detection and response guardrails?
5. Does it define acceptance criteria before launch, including sample data, test cases, approval path, audit log, error branch, and rollback?
6. Does the final recommendation name one first safe build, one immediate next action, one premature move to avoid, and an expansion trigger?
