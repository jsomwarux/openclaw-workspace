# high-stakes-draft-eval Autoresearch Checklist

Answer yes/no for each evaluated draft review:
1. Did it check specificity against concrete signals/facts rather than generic language?
2. Did it verify source freshness and flag stale/cached evidence risk?
3. Did it evaluate ICP fit and buyer-world language?
4. Did it check proof/credibility without unsupported claims?
5. Did it check CTA quality, voice rules, and banned outreach patterns?
6. Did it return a clear SHIP / FIX THEN SHIP / HOLD verdict with exact edits when needed?
