# Autoresearch Checklist — notion-integration

Score Notion writes/calendar pushes against these yes/no checks:

1. Does it use the correct Notion database and property names?
2. Does it include the required Drive link when content or drafts are involved?
3. Does it avoid duplicate calendar/task entries for the same post or item?
4. Does it set date, platform/type/status fields correctly?
5. Does it verify the Notion API write response before claiming success?
6. Does it batch writes where appropriate to avoid rate-limit loops?
