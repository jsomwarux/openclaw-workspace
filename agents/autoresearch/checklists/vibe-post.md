# Autoresearch Checklist — vibe-post

Score vibe-post.py behavior with yes/no answers:

1. Does it select only approved, unposted TikTok queue entries for the requested product?
2. Does slide parsing fail closed with a clear error instead of silently posting malformed text?
3. Does it use the correct product photo/screenshot library and avoid unsupported/fabricated assets?
4. Does it mark queue/performance state consistently only after post creation succeeds?
5. Does it notify JT with the post result while avoiding duplicate sends or misleading success messages?
6. Does it preserve product-specific safety rules, especially no unsupported medical, financial, or fake relationship claims?
