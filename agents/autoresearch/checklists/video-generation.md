# Autoresearch Checklist — video-generation

Score video generation workflows against these yes/no checks:

1. Does it choose a format that matches the product/message and target platform?
2. Does the script/hook make the value obvious in the first 3 seconds?
3. Does it use concrete product metadata, proof points, or niche pain instead of generic ad copy?
4. Does it generate or queue the expected asset successfully?
5. Does it include posting/queue metadata and next action for JT or the agent?
6. Does it avoid unsupported claims, copyrighted assets, or unsafe media prompts?
