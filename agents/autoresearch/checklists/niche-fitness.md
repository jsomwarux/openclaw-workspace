# Autoresearch Checklist — niche-fitness

Score niche-fitness evaluations against these yes/no checks:

1. Does it evaluate niche pain, budget, accessibility, urgency, and JT fit separately?
2. Does it include evidence from recent market/prospect signals?
3. Does it avoid promoting niches solely because they are trendy?
4. Does it produce a clear pursue/defer/monitor recommendation?
5. Does it define a specific trigger for deferred niches?
6. Does it update the appropriate memory or future-signal file when decisions change?
