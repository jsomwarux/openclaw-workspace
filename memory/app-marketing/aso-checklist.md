# App Marketing OS — ASO Checklist

## Purpose
Improve App Store visibility and conversion for Vista and future iOS apps.

## Checklist
For each App Store app:

### Metadata
- App name
- Subtitle
- Keyword field
- Category
- Description first 3 lines
- Full description
- Promotional text

### Screenshots
- First screenshot hook
- Second screenshot proof/value
- Third screenshot feature clarity
- Are screenshots readable at small size?
- Does the first screenshot explain why someone should care?

### Competitor Scan
- Top 5 competing apps
- Their title/subtitle keywords
- Screenshot hooks
- Review themes
- Weaknesses users complain about

### Conversion Assets
- App preview video if worth it
- Review prompt timing
- Landing page link consistency
- Support URL
- Privacy URL

## Vista — Initial ASO Focus
Priority: high

Hypotheses:
- Current highest-intent keywords likely include: movie tracker, movie rating, Letterboxd alternative, film tracker, movie list, movie recommendations, movie diary.
- Vista’s differentiator should emphasize 1–100 precision rating and taste profile.

First action:
- Capture current App Store title/subtitle/screenshots/description.
- Compare against Letterboxd, IMDb, Serializd/film-tracking alternatives, and small indie movie trackers.
- Recommend one metadata test and one screenshot test.

## Future Apps
Any future iOS app must complete ASO review before launch week.
