# App Marketing OS — SEO Backlog

## Purpose
Create durable search-discovery pages for apps so visibility is not dependent only on TikTok/X/Reddit.

## Page Types
- Alternative/comparison page: `[App] vs [Known Tool]` or `[Known Tool] alternative`.
- Use-case page: `best app for [specific job]`.
- Methodology page: explain the framework behind the product.
- Glossary/education page: rank for niche problem terms.
- Launch/waitlist page: capture demand before product is complete.
