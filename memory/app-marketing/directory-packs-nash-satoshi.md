## 150-word description
Nash Satoshi is a crypto research app that ranks tokens using a 4-model AI analysis framework. Each token is evaluated across narrative strength, incentive design, coordination dynamics, liquidity context, reflexivity, and game-theory positioning. The system compares model consensus and disagreement so users can see where signals are stronger and where assumptions are fragile.

The product is not a price predictor, buy/sell signal, or financial advice tool. It is a research layer for crypto users who want to understand token positioning beyond momentum charts and single-model summaries.

Nash Satoshi includes rankings, token scorecards, model reasoning, and a public methodology page explaining how the scoring system works.

I built Nash Satoshi because most crypto ranking tools either over-index on price action or hide the reasoning behind the score. I wanted a research layer that shows how multiple AI models evaluate token positioning and where they disagree.
