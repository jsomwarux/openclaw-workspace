# App Marketing OS — App Registry

## Status Values
- `active`: product is live enough for ongoing marketing.
- `prelaunch`: product is not live but should build audience/waitlist.
- `pending`: product exists but marketing should wait for a blocker to clear.
- `paused`: product should not receive routine marketing.
- `idea`: passive-income idea not yet selected for build.
