## 2026-05-13 heartbeat synthesis — property ops wedge sharpening

Live search was unavailable during this heartbeat, so this pass used saved recent research rather than treating provider errors as market signal.

Current strongest property-management wedge for JT remains **exception visibility**, not generic AI:
- Maintenance triage: scattered tenant requests → urgency classification → tenant/vendor reply drafts → overdue follow-up list.
- COI/insurance compliance: expiring vendor/tenant proof → missing document queue → approver-visible follow-up trail.
- Owner reporting: weekly summary of stuck items, approvals needed, and compliance exposure.

Sharper offer language:
> “I build a small command center for property operators: what changed, what’s stuck, who owns the follow-up, and what needs approval — using the inbox/spreadsheets/shared drives you already have.”

Next useful outbound angle: pair maintenance + COI as one “property ops exceptions” audit, then let the prospect choose the painful workflow for a 7-day prototype.
