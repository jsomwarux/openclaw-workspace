# Enterprise Platforms Beyond Agentforce — JT Somwaru Consulting Expansion Research
*Researched: 2026-02-28 | Eve*

---

## Context
JT Somwaru Consulting currently targets Salesforce/Agentforce shops. This research evaluates other platforms with AI/automation extension points where JT can build consulting services.

---

## Top 3 Picks (Ranked)

### 1. 🥇 HubSpot — AI CRM + Workflow Automation
- **Who uses it:** Huge SMB + mid-market CRM base. Tons of NYC businesses already on it.
- **AI extension points:** Unified AI across marketing, sales, and service. Agents for lead sourcing, qualification, outreach. Strong API.
- **Consulting angle:** Migrate or optimize existing HubSpot workflows + add AI agents on top. Lower technical barrier than Salesforce. Faster implementation → better margin on smaller projects.
- **Overlap with JT Somwaru Consulting niches:** Real estate brokers, insurance, skilled trades all commonly run HubSpot.
- **Gap vs. Agentforce:** No certification ecosystem yet, so JT's general AI consulting advantage holds. Lower competition than Salesforce partner market.
- **Priority:** 🔴 High — consider adding as a primary JT Somwaru Consulting platform.

### 2. 🥈 Notion Custom Agents — Team Workflow Automation
- **Who uses it:** Knowledge workers, ops teams, growing startups + mid-size companies. Ramp, Remote, Braintrust confirmed deployments.
- **AI extension points:** Custom Agents launched 2026 — automate recurring team work 24/7. Reads/writes Notion DB, runs on schedule or trigger.
- **Consulting angle:** Build custom agents for specific ops pain points (e.g., property management lease tracking, construction RFI logging, insurance claim intake). Free until May 3, 2026 → strong "free pilot" pitch.
- **JT's edge:** Can build + deploy a Notion agent faster than internal teams who have no AI background.
- **Priority:** 🟠 High — test with one client (Aya could use this for co-living ops).

### 3. 🥉 Airtable Superagent — Multi-Agent Data Automation
- **Who uses it:** Data-heavy ops teams. Construction, property management, wholesale — all track data in Airtable.
- **AI extension points:** Superagent launch (Jan 2026) — multi-step agent chains with full execution visibility. Key insight: performance depends on data structure quality > model selection. JT can design the schema = highest value-add.
- **Consulting angle:** "AI-ready database architecture" — set up Airtable bases correctly so AI agents can run on them. Premium service: most companies' Airtable is a mess.
- **Priority:** 🟡 Medium — niche use case, but very defensible expertise.
