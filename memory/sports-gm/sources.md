# Sports GM Reference Sources

## Market Price
- KeepTradeCut — dynasty value/trend sentiment
- FantasyCalc — trade database / calculator
- DynastyProcess — calculator baseline
- Dynasty Data Lab — ADP / rookie ADP where available
- Sleeper — ADP, rosters, league context where available
- DLF/FantasyPros/Rotoballer charts — secondary market references
- X/Reddit — sentiment only, never truth

## Football Signal
- PlayerProfiler — athletic/production profile
- PFF — grade and usage context where accessible
- FantasyPoints Data / FTN / ETR concepts — advanced usage where accessible
- Reception Perception — WR route/coverage profile where accessible
- NFL Next Gen Stats
- Pro Football Reference / Stathead
- Team depth charts, contracts, draft capital, coaching context
- Vegas totals/player props — market-implied expectations

## Framework Inputs
- Position age curves
- Draft capital hit rates
- Rookie pick hit rates by slot/tier
- Positional replacement value
- League format scarcity: Superflex, TE premium, start size, bench depth
- Historical ADP/value movement after injuries, rookie hype, role changes, playoff production
