Body:

I’ve been working on Nash Satoshi, a crypto ranking app that scores tokens with multiple AI models instead of one model output.

The part I kept running into: an “AI score” is not very useful if no one can see what the score is supposed to measure.

So I published the methodology page first instead of only pushing the rankings page.

The framework looks at:

- narrative strength
- incentive design
- coordination dynamics
- liquidity context
- game-theory positioning
- model consensus and disagreement

The important part is disagreement.

If Claude likes the narrative, Gemini flags liquidity, and Grok pushes back on coordination, that is more useful than pretending there is one clean answer.

It is not meant to predict price or tell anyone what to buy.

It is a research layer for seeing which tokens have stronger positioning and where the assumptions are fragile.

Curious how people here would improve this kind of methodology. What would you want disclosed before trusting an AI-generated crypto ranking?
