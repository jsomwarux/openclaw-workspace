An AI agent should not be your first design decision.

The first decision is where the work changes hands.

If the handoff is vague, the agent will inherit the mess. If the handoff is clear, the agent has a job: read the input, make the decision, route the next step, escalate the edge case.

Better agents start with narrower workflows.
