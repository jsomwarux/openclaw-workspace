
## 09:30 - Coding agent spawns 70 sub-agents from one prompt
**Status:** Ready to post
**Hook:** Agents that can self-spawn need budget ceilings before more autonomy.
**Source:** https://aiweekly.co/alerts/claude-code-autonomously-spawns-70-research-agents
**Pattern inputs:** concrete incident hook, specific number proof, practitioner safeguard, direct owner/audit implication.
**X Thread:**
A coding agent reportedly spawned ~70 sub-agents from one deep-search prompt. That changes the risk profile: your agent can design a workflow you never asked it to build.
---
The run split itself into 4 phases and parallel research agents. The report says the mode had no built-in spend cap, so the first warning can be the bill.
---
For your business, agent access needs a ceiling before it needs more autonomy: budget cap, max sub-agents, owner, stop rule, and audit log. (via @AIWeekly)
