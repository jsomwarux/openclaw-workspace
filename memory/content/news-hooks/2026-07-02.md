
## 09:30 - Org-wide MCP connector provisioning moves agent tooling into IT identity layer
**Status:** Ready to post
**Hook:** Agent connectors are becoming IT-managed infrastructure with centralized provisioning and authorization.
**Source:** https://releasebot.io/updates/anthropic/claude
**X Thread:**
A major AI lab just moved MCP tool connectors into org-wide admin provisioning. The immediate implication: agent tooling now sits in the IT identity layer. (via @releasebot)
---
Admins can provision connectors through an identity provider. Users get access on first login. Authorization stays centrally configured across the org.
---
For SMB operators, this changes the build checklist. Before an agent touches docs, email, CRM, or files, your team needs connector ownership, permission rules, and a review path.
